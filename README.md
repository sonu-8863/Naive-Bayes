#❤️ Heart attack possibility Prediction using Gaussian Naive Bayes 

This project focuses on predicting the presence of heart disease using a machine learning classification model. The dataset was loaded and processed using Pandas, where the target variable was separated from the input features.

The data was split into training and testing sets using train_test_split to ensure proper model evaluation. A Gaussian Naive Bayes classifier was implemented due to its efficiency and suitability for numerical data.

The model was trained on the training dataset and evaluated on the test dataset using key performance metrics.

📊 Model Performance
Accuracy: 86.88%
Precision: 0.90
Recall: 0.84
🔍 Key Steps Performed
Data loading and preprocessing
Feature-target separation
Train-test splitting
Model training using GaussianNB
Model evaluation using accuracy, precision, and recall
